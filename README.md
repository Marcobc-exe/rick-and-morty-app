Consume the Rick and Morty API and push data to the front-end to create cards characters

# Getting Started 
First, run the development server: `npm start` <br>
Open http://localhost:3000 with your browser to see the result.

### Technologies
- React.js
- Bootstrap

### Requirements
- Node v18.7.0
- npm v9.2.0
